Integrasi Sinkronisasi Google Sheets

Secara default, aplikasi berjalan menggunakan penyimpanan lokal browser (LocalStorage). Untuk menyimpan data secara real-time ke spreadsheet Google Anda, ikuti panduan berikut:

Langkah-Langkah:
  1. Buat Spreadsheet Google Baru di Drive Anda.
  2. Buka menu Ekstensi > Apps Script.
  3. Hapus semua kode bawaan, lalu salin kode backend Apps Script yang ada di bawah ini.
  4. Klik tombol Terapkan / Deploy > Penerapan Baru. Pilih jenis: Aplikasi Web.
  5. Ubah akses menjadi: "Siapa saja" (Anyone), lalu klik Deploy.
  6. Salin URL Aplikasi Web yang dihasilkan dan tempel pada kolom input di bawah ini.
Klik di dalam kotak teks di atas untuk otomatis memilih semua kode lalu tekan Ctrl+C.
